## Description

Barramento de API's do projeto Consult.

## Installation

Primeiro passo:
```bash
$ npm install
```
Segundo passo:
```bash
$ docker compose up -d
```
Terceiro passo:
```bash
Acesse: [localhost](http://localhost:8080) e crie uma nova Database chamada "Consult"
```
Quarto passo:
```bash
$ npx prisma migrate dev
```

## Running the app

```bash
$ npm run start:dev
```
